# 주제 선정 기록

## 선정된 주제

- 제목: DecisionOps Control Tower
- 문제: 운영 ML 예측과 agentic review 결과를 하나의 API·dashboard·review queue로 배포 가능한 의사결정 제품으로 연결한다.
- 대상 의사결정: demo mode 가능 여부, public deploy 여부, reviewer approval queue 우선순위.
- 최종 사용자/운영자: 운영 책임자, ML product owner, reviewer, hiring reviewer.

## 후보 평가

토요일 seed 단계에서 최소 5개 후보를 비교합니다.

| 후보 | 도메인 | 데이터 가능성 | 운영/제품 가치 | 연구 난이도 | 채용시장 신호 | 판단 |
|---|---|---|---|---|---|---|
| DecisionOps Control Tower | 운영 ML + agentic review | Stage 1/2 산출물 존재 | 최종 납품 신호 강함 | 높음 | MLE/Product/Applied AI | 선정 |
| Generic LLMOps dashboard | LLM traces | 새 데이터 필요 | 일반적 | 중간 | Applied AI | 보류 |
| Mobility command center | bike-share only | Stage 1 산출물 존재 | 도메인 좁음 | 중간 | DS/Product DS | 보류 |
| Incident response dashboard | NY 511 only | public data 가능 | 교통 특화 | 중간 | DS/Analytics | 보류 |
| Planner-style ops app | 업무 관리 | 기존 app 경험 있음 | 포트폴리오 연결 약함 | 낮음 | Product Engineer | 보류 |
